Status Enums
============

Status enums for tracking execution and job states.

Overview
--------

Routilux provides several enum types for tracking the state of various
workflow components:

* ``ExecutionStatus``: Status of routine executions
* ``JobStatus``: Status of jobs
* ``RoutineStatus``: Status of routines within a worker
